# Grocery Profit & Loss Tracker

## What I Did:
- **Inter-sheet Linking:** Connected Purchase and Sales sheets so data flows automatically without manual typing
- **Automated Profit Logic:** Smart formula to calculate Sales Price by automatically adding 70% markup to Purchase price
- **Expense Management:** Tracked hidden costs (carriage, loading, rent, electricity) to get real "Total Expense"
- **Profit & Loss Analysis:** Final dashboard showing exact profit/loss on each grocery item

## Tools Used:
Microsoft Excel (Inter-sheet Data Linking, Business Logic Formulas, Expense Tracking)

- Multi-sheet Automation
- Cost Analysis
